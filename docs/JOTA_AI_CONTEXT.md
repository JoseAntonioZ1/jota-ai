# JOTA AI

## Autor

José Antonio de la Cruz Portal

---

# Descripción General

JOTA AI es un asistente inteligente multimodal con avatar virtual diseñado para mejorar la autonomía digital de adultos mayores.

El sistema permitirá interactuar mediante:

- Voz
- Texto
- Avatar animado

La inspiración conceptual proviene de JARVIS de Iron Man, adaptado a necesidades reales de accesibilidad y asistencia tecnológica.

---

# Problema

Muchos adultos mayores presentan dificultades para utilizar smartphones debido a:

- Falta de experiencia tecnológica.
- Problemas visuales.
- Problemas de memoria.
- Problemas de motricidad.
- Interfaces complejas.

Como consecuencia:

- Dependencia de familiares.
- Baja adopción tecnológica.
- Dificultad para acceder a servicios digitales.

---

# Objetivo Principal

Incrementar la autonomía digital de adultos mayores mediante un asistente inteligente capaz de comprender lenguaje natural y proporcionar asistencia contextual.

---

# Visión

Crear un asistente inteligente que funcione como un compañero digital permanente.

El usuario debe sentir que interactúa con una entidad amigable y confiable, no con una aplicación tradicional.

JOTA debe ser:

# Avatar

El avatar debe representar visualmente a JOTA.

Características:

- Apariencia amigable.
- Estilo moderno.
- Expresiones faciales.
- Animaciones suaves.
- Presencia constante dentro de la aplicación.

Estados mínimos:

- Escuchando.
- Pensando.
- Hablando.
- Esperando.
- Feliz.
- Confundido.
- Preocupado.
